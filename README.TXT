Pour lancer l'appli:
- cloner le projet
- npm install
- ionic platform add android
- ionic run android

fonctionnalités ajoutées:
- dans la page qui retrace le parcours perso, les sessions sont triées. 
  je n'ai pas eu le temps de tester le chevauchement des dates mais j'ai une implémentation.
- Dans la page des sessions, elle sont triées de la même façon que dans la page parcours.
  Les sessions du parcours sont affichées en vert pour indiqué a l'utilisateur qu'il l'a selectionné.
- Cette selection s'effectue dans la page detail d'une session avec un bouton toogle.
- j'ai aussi ajouter la possiblité de filter les sessions et les présentateurs.

j'ai quelques soucis aux niveau CSS.
